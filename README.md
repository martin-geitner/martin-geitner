## Projekte

### 🔌 [llm-backend-api](https://github.com/martin-geitner/llm-backend-api)
> REST-API-Backend zur Anbindung lokal betriebener Sprachmodelle

FastAPI-basierter Backend-Service, der ein lokal laufendes Mistral-Modell via Ollama anbindet. Entwickelt mit dem Fokus auf on-premise-Betrieb und Datensouveränität.

- Konzeption und Implementierung einer REST-Schnittstelle (JSON, HTTP, OpenAPI)
- Request-Response-Pipeline: HTTP-Anfrage → FastAPI → Ollama-API → Mistral → Antwort
- Prompt-Engineering-Techniken: System-Prompts, Few-Shot, Chain-of-Thought
- Unit-Tests mit **pytest**, CI-Setup mit **GitHub Actions**
- `Python` `FastAPI` `Ollama` `Mistral` `pytest` `OpenAPI`
